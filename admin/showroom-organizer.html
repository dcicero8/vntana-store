Showroom Organizer

Paste a showroom link (or its ID) and click Organize. Assets are grouped by style and sorted by color code, automatically.

For this to work, every asset in the showroom needs:
1. A Style attribute (this becomes the group name)
2. A file name like STYLE_COLOR.glb (for example 41795_CVMI.glb)

If the showroom won't organize, an asset is usually missing its Style attribute or isn't named correctly. Fix it in VNTANA and try again, or send the showroom link to Derek.
