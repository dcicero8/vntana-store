    How to customize the VNTANA Viewer

    The viewer on the left is a single <vntana-viewer> element. Everything else , the product page, controls, and interactions , is our code. VNTANA handles the 3D, AR, and CDN hosting.

    viewer.poster

    Branded loading screen

    Product thumbnail shown clearly at 35% opacity against the viewer background , a clean "preview while loading" rather than a blurred overlay. Fades out when the viewer fires its load event, with a 14 s safety fallback.

    viewer.poster = thumbnailUrl
    <vntana-hotspot>

    Custom hotspot pins

    38 px pins in brand blue with a pulsing ring animation. Clicking any pin snaps the camera to the platform-configured angle via setCameraRotation().

    const h = document.createElement("vntana-hotspot") h.position = dimensions.position h.normal = dimensions.normal viewer.append(h)
    setCameraRotation()

    Camera angle presets

    Front, Side, and Detail snap to Euler angle strings , the same format VNTANA uses internally. Reset re-applies the exact camera state from the platform.

    viewer.setCameraRotation("0rad 1.5707963rad 0rad") // side view
    viewer.environmentSrc

    Lighting switcher

    One property swap exchanges the HDR environment map live , Studio, Neutral, or Dim , no model reload. All HDRs are hosted on VNTANA's CDN.

    viewer.environmentSrc = "https://storage.googleapis.com/.../Indoor_Studio_2.hdr"
    viewer.background

    Background toggle

    Accepts any hex colour or CSS gradient. Switches the canvas background without touching the model or lighting rig.

    viewer.background = "radial-gradient(#484848, #000000)"
    Public API

    Render gallery

    Thumbnail pulled from VNTANA's public CDN , no auth token required. AI renders from the Gemini pipeline load as plain <img> tags.

    GET /assets/organizations/{org}/clients/{ws}/thumbnail/{blobId}
    platform

    360° Turntable

    The platform generates a 120-frame 4K turntable directly from the 3D model, no photography needed. Exported as MP4 and GIF and attached to the product automatically.

    Gemini AI

    AI enhanced renders

    A custom n8n pipeline listens for Live Public events and fires Gemini to generate lifestyle renders from the platform thumbnail , automatically, no manual prompting required.

    VNTANA webhook → n8n → Gemini imageGenerate → gallery
    enableAutoRotate

    Auto-rotate toggle

    One boolean flips the spin on or off at any time , useful for drawing attention on page load or letting a user pause to inspect a detail. Platform config is intentionally overridden here so the PDP starts still.

    viewer.enableAutoRotate = true // start spinning viewer.enableAutoRotate = false // pause
    shadowIntensity

    Shadow intensity slider

    Live range input drives the ground shadow from invisible to full. Any numeric property on the viewer can be wired to a UI control this way , no reload, no re-fetch.

    viewer.shadowIntensity = 0.45 // 0 – 1
    canvas.toBlob()

    Capture view

    Grabs the current frame directly from the WebGL canvas in the viewer's shadow DOM and triggers a PNG download, whatever angle, lighting, and background the shopper has set.

    scene.explodedStrength

    Guided tour

    Hotspot camera angles and explode states are stored in the platform and fetched at runtime. The tour steps through every pin in sequence , camera snap, popup, explode if configured , all from a single async loop.

    viewer.setCameraRotation(hotspot._camera.cameraRotation) viewer.scene.explodedStrength = hotspot._explode.explodedStrength
