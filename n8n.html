Integrations

VNTANA + n8n

Automate product attribute updates, hotspot translations, and file management using the VNTANA Platform API — no code required beyond configuring a workflow.

Powered by n8n

How the integration works

Data flows from a trigger on the left, through VNTANA API operations in the middle, to live product updates on the right.

📥

Inputs

📄

Spreadsheet Upload

Drop an .xlsx into the admin UI. Parsed locally, sent to n8n via webhook.

Webhook trigger
📎

Product Attachment

Attach a spreadsheet directly to a VNTANA product. n8n detects it automatically.

Timer trigger
⏱️

Scheduled Poll

Runs every 15 minutes. Checks for new attachments matching a filename pattern.

Schedule trigger
🖱️

Manual Sync Button

One-click from the admin page triggers n8n to re-read the attached file and update immediately.

Webhook trigger
⚙️

Operations

🔐

Authenticate

Posts credentials to /v1/auth/login and passes the token to all subsequent requests via X-AUTH-TOKEN.

Platform API
🔍

Search Attachments

Queries /v1/attachments/search for files on the product. Filters by filename prefix and skips already-processed UUIDs.

Platform API
📊

Parse Spreadsheet

n8n's built-in Spreadsheet File node reads the .xlsx and maps each row to JSON. Expects two columns: attribute_key and attribute_value.

n8n native
🔄

Build Attributes Array

Transforms spreadsheet rows into the [{name, value}] format the VNTANA Products API expects.

Code node
📎

Upload as Attachment

Posts the raw file to /v1/attachments/upload/product-attachment, then registers the blobId on the product.

Platform API
📤

Outputs

✅

Product Attributes Updated

All attributes on the VNTANA product are replaced with the values from the spreadsheet. Live immediately — no publish step.

VNTANA Platform
🌍

Multilingual Hotspot Text

Translations are stored as hotspot_translations_fr etc. attributes and read at page load — no CMS, no re-deploy.

Live on viewer
📁

File Stored on Product

The spreadsheet lives as an attachment in VNTANA alongside the 3D asset — auditable, replaceable, no external storage needed.

VNTANA Platform
↩️

Webhook Response

n8n responds to the admin UI with success status and the languages that were updated, so the operator gets immediate confirmation.
